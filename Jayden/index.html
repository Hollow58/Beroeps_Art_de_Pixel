Temple Run

Temple Run is een actie-avonturengame waarin je als avonturier door een oud tempelcomplex rent. Je moet obstakels en valstrikken ontwijken terwijl je munten verzamelt en demonen ontvlucht. De snelle, verslavende gameplay en levendige graphics maakte het een populaire mobiele game.
